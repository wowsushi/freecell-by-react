# Freecell
新接龍小遊戲

## 使用技術
- React
- HTML5 Drag and Drop API
- CSS
- JavaScript

## 功能
- 可以玩新接龍遊戲 (目前僅支援單張拖曳)
- 順利排完會跳出勝利畫面引導玩新關卡
- 可以返回上一步
- 可以暫停遊戲
- 遊戲會計時與記分
- 可以重玩關卡
- 可以選擇玩新關卡
